# HackStorm SHOP Discord Bot

Команды бота полностью через slash-команды Discord:

- `/verify` — отправляет панель верификации. Кнопка выдаёт роль `1429401945671467028`.
- `/partners` — отправляет панель партнёров.
- `/partnersadd` — добавляет партнёра в `database/partners.json`.
- `/nabor` — отправляет панель поддержки. Пользователь создаёт приватный тикет.

Удалены старые системы:

- `/battlepass`
- `/anticrash`
- старые prefix-команды через `!`

## Установка

1. Удали старые файлы бота на хостинге.
2. Загрузи содержимое этого архива.
3. Установи зависимость:

```txt
discord.py==2.5.2
```

4. Токен вставь в панель хостинга. Dotenv не используется.
5. Если панель хостинга не передаёт токен автоматически, создай файл `token.txt` рядом с `main.py` и вставь туда только токен без кавычек и без слова `Bot`.
6. Запусти `main.py`.

## Важные права бота

У бота должны быть права:

- Manage Channels
- Manage Roles
- Send Messages
- Embed Links
- Attach Files
- Read Message History
- Use Slash Commands

В Discord Developer Portal включи:

- Server Members Intent

Message Content Intent не нужен, потому что команды работают через `/`.

## Тикеты

Команда `/nabor` публикует панель поддержки. При нажатии кнопки создаётся приватный канал вида:

```txt
support-nickname
```

Канал видят только:

- пользователь, который создал тикет;
- роли поддержки из `support_role_ids`;
- бот.

Внутри тикета есть только одна кнопка:

```txt
Удалить тикет
```

Она удаляет канал через 3 секунды.
